# Build Log

This file tracks project builds and their outcomes.

## Format

Each build entry follows this format:

    ## Version X.Y.Z - YYYY-MM-DD HH:MM:SS UTC

    - **Build type**: patch/minor/major
    - **Status**: success/failure
    - **Changes**: Brief description of what changed
    - **Outputs**: What was generated/archived
