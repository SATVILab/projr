# Show a standard informational message

Show a standard informational message

## Usage

``` r
.cli_message(
  ...,
  log_type,
  console_level = "std",
  console_fn,
  output_level = "std",
  .envir = parent.frame()
)
```

## Arguments

- ...:

  Message components passed to cli::cli_alert_info

- output_level:

  Character. Current output level.

- .envir:

  Environment for variable evaluation
