# Show a step message (sub-item in a stage)

Show a step message (sub-item in a stage)

## Usage

``` r
.cli_step(..., output_level = "std", .envir = parent.frame(), log_file = NULL)
```

## Arguments

- ...:

  Message components passed to cli::cli_li

- output_level:

  Character. Current output level.

- .envir:

  Environment for variable evaluation

- log_file:

  Character. Path to log file (optional).
